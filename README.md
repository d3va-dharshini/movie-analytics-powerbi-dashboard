# 🎬 Movie Industry Performance Dashboard

This project is a Power BI dashboard created to analyze movie industry data and understand trends in profit, ratings, and overall performance.

---

## 📊 Overview

The dashboard provides insights into:
- Total number of movies and overall profit  
- Maximum IMDb rating  
- Profit trends over different years  
- Comparison between movie industries  
- Performance of studios and individual movies  

---

## 🛠 Tools Used

- Power BI  
- Data Visualization  
- Basic Data Analysis  

---

## 📈 Key Insights

- Hollywood generates higher total profit compared to Bollywood  
- Certain years show a noticeable increase in movie profits  
- Some studios consistently produce higher-performing movies  
- Language plays a role in movie distribution and performance
